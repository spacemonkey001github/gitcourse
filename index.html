Github

Github is the industry standard collaboration tool that helps two or more individuals work on the same project without the need to manually coping and pasting the necessary files among them. It also helps in making sure that the files being worked on are updated all the time.

Git Bash Commands

git clone

To clone the repositories from github.

git status

To check the status of the files.

git add

To upload the untracked files to github.

git commit -m "message"

To commit here means that you are sure of the files uploaded with "git add" and now are to be published in github.

git push

To let the github.com knows the changes.

git pull

To download the latest update of the projectfrom github.com

git branch

list the branches of the project

git branch name

to create a new branch.

git checkout name

To switch the branch.
